## Project Overview

A lightweight, browser-based LaTeX editing environment that runs entirely on the user's local machine. Similar to Overleaf but with zero cloud dependency.

**Core Features:**
- Edit LaTeX documents in the browser (Monaco Editor)
- Compile documents locally to PDF
- Preview generated PDF in the browser
- Create new projects and import existing ones
- Work with .tex, .bib, images, and other assets

**Tech Stack:**
- Frontend: React 18 + TypeScript + Vite + Tailwind CSS + Monaco Editor
- Backend: Node.js + Express + TypeScript
- PDF: PDF.js (react-pdf)
- State: Zustand
- Monorepo: Turborepo + pnpm

## Development Notes

### Prerequisites:
- Node.js 18+ 
- pnpm 8+
- LaTeX distribution (TeX Live, MiKTeX, or MacTeX)

### Development Workflow:
1. Run `pnpm install` to install dependencies
2. Run `pnpm dev` to start both frontend and backend
3. Frontend: http://localhost:3000
4. Backend: http://localhost:3001

### Module Boundaries (Enforced via ESLint):
- Features cannot import from other Features
- Components can only import from Utils/Types
- Services can only import from Utils/Types
- Utils can only import from Types
- Types cannot import from anything

### Security Rules:
- Bind to localhost only (127.0.0.1)
- LaTeX compilation runs with user-level privileges
- User documents never leave local machine
- Disable LaTeX \write18 for security
